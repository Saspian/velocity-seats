# High-Fidelity Ticket Booking UI

An incredibly high-fidelity, clean, and modern web application UI for an online concert ticket booking system during a flash sale.
A real-time high-concurrency ticket booking system using a Node.js backend, a Redis distributed locking mechanism, and PostgreSQL for final transactions.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
